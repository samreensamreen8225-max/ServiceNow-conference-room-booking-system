# ServiceNow Conference Room Booking System

## Project Overview

This project is a Conference Room Booking System developed in ServiceNow using a custom table, Record Producer, and Business Rules.

Employees can submit conference room booking requests through a Record Producer form. The submitted data is automatically stored in a custom Conference Room Booking table, where room availability is validated and booking status is updated automatically.

## Features

* Custom Conference Room Booking table
* Record Producer for booking requests
* Automatic record creation
* Employee Name auto-population
* Department auto-population
* Booking status management
* Room availability validation
* Automatic Approved / Rejected status updates
* ServiceNow platform implementation

## Technologies Used

* ServiceNow
* Record Producers
* Business Rules
* Custom Tables
* Dictionary Entries
* Form Design

## Workflow

1. Employee opens the Conference Room Booking form.
2. Employee enters meeting details.
3. Record Producer creates a booking record.
4. Booking data is stored in the Conference Room Booking table.
5. Business Rule checks room availability.
6. Booking status is automatically set to Approved or Rejected.
